# Dicoding Dashboard 

## How to run Stream lit in Visual Studio Code
```
Buka Folder Yang Ingin Dijadikan Path ( Kasus Saya nama folder nya dashboard )
Masuk ke Directory Folder lewat Terminal Visual studio code ( cd dashboard )
jalankan di terminal yang sama python -m venv .venv
jalankan di terminal yang sama .\.venv\Scripts\Activate.ps1
install library pip install streamlit pandas matplotlib seaborn
jalankan streamlit
```
